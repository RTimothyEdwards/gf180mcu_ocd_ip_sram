Open Circuit Design 3.3V SRAM macros for GlobalFoundries GF180MCU open PDK
===========================================================================

This repository contains 3.3V SRAM macros as an extension to
`Google's open source PDK for GlobalFoundries 180nm MCU process node <https://github.com/google/gf180mcu-pdk>`_.
Upstream libraries are currently owned by FOSSi Foundation <https://github.com/fossi-foundation>
